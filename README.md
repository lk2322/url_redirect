# url_redirect
Перенаправление ссылок 

## Установка
```
git clone https://github.com/lk2322/url_redirect.git
cd url_redirect
pip3 install -r requirements.txt
```
## Настройка
1. Открыть `main.py`
2. В поле `DOMAIN` ввести свой домен(ip), в поле `HOST_IP` ip устройства

## Использование
```
python main.py 
```
